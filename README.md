# Infinite EduSuite

A modern education management platform built with Vite, React, TanStack Router, Supabase, and Cloudflare tooling.

## Features

- Student portal and admin dashboard
- Role-based access control (RBAC)
- Supabase integration for authentication and database
- Vite-powered development workflow
- Built-in utilities for finance, ledger, and notifications

## Getting Started

### Prerequisites

- Node.js 20+ or Bun
- npm or bun
- Supabase project and environment variables configured in `.env`

### Install

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

### Lint and format

```bash
npm run lint
npm run format
```

## Notes

- Keep sensitive values in `.env`; this file is ignored by Git.
- The repository remote is configured to `https://github.com/Semendotech/Infinite-Edusuite.git`.
